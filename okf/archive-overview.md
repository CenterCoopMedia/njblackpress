# Archive overview

The archive documents Black-owned and Black-focused publications connected to New Jersey from 1880 to the present. It includes newspapers, magazines, newsletters, journals, and digital outlets.

## Wiki structure

- One publication record per file under `publications/`.
- City, decade, and format pages group records for browsing.
- `data-model.md` explains the fields used in each publication page.
- `log.md` records generation notes.
