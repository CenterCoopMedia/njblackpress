# Data model

Publication pages are generated from `data/publications.json`. The source values are preserved as written except for empty values, which are shown as unknown in tables.

## Fields

- `id`
- `name`
- `alternateName`
- `city`
- `publishers`
- `yearFounded`
- `yearCeased`
- `frequency`
- `format`
- `languages`
- `archiveUrl`
- `websiteUrl`
- `targetAudience`
- `primaryFocus`
- `medium`
- `missionStatement`
- `keyStaff`
- `historicalNotes`
- `isActive`
- `decade`
